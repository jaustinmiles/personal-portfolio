Master's Project: Infinite Procedural Content Generation Dev Tool

Developer tool for creating procedurally generated terrains, fully integrated with the Unity Editor. Users define graphs with logic nodes for modifying the world. Examples include: introducing perlin noise octaves, selecting attributes based on biomes, normalizing terrain values, and applying erosion.

Out of a template terrain, as shown in the picture, users can generate seamless tiles endlessly out of the template tile during runtime
